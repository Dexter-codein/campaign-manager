# Campaign manager Dashboard in React JS

React dashboard for campaign management. [Live Demo](https://soorajchandran.github.io/image-hover-zoom/)


Clone the repo

```sh
$ npm install
$ npm start
```

### Linting


```sh
$ yarn run lint
```


### Use scss

```sh
$ npm run watch-css
```
