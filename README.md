# Campaign manager Dashboard in React JS

React dashboard for campaign management. [Live Demo](http://campaign-manager-project.s3-website-us-west-2.amazonaws.com/)


Clone the repo

```sh
$ npm install
$ npm start
```

### Linting


```sh
$ yarn run lint
```


### Use scss

```sh
$ npm run watch-css
```
